Quem criou o HTML?

Voltando há algumas poucas décadas… O HTML, ou HyperText Markup Language, foi criado em 1990 pelo cientista Tim Berners-Lee, enquanto ele trabalhava no Centro Europeu de Pesquisa Nuclear, na Suíça.

O objetivo dele era criar uma maneira de compartilhar e trocar documentos pesquisas entre cientistas de diferentes universidades. Naquela época, a linguagem não era uma especificação, mas sim uma forma de comunicação entre a comunidade científica.

Contudo, a prática combinada com a internet pública tornou-se um sucesso mundial. Além disso, Berners-Lee criou outros protocolos associados como o HTTP e WWW (World Wide Web). Ao longo dos anos, o HTML foi evoluindo para se tornar a linguagem de marcação padrão para páginas da Web.

Na época, o HTML foi lançado pelo W3C, sendo a comunidade que normatiza e desenvolve padrões para como devem ser as coisas na Internet.

Oque são Clients

(Voltar)

Os clients são as pessoas que consomem a informação através de dispositivos, essas informações são encaminhadas dos servidores ou serves que entre outras coisas hospedam os sites que são acessados.

Oque são Browsers

(Voltar)

Os Navegadores ou browsers são aplicativos que são instalados nos dispositivos com objetivo de interpretar os arquivos enviados pelos servidores. A maioria dos navegadores atualmente são gratuitos que seguem os padrões estabelecidos pela W3C. O primeiro navegador foi MOSAIC e o Internet Explore foi durante muito tempo o navegador mais usado, devido está presente em quase todos os computadores. Aplicações Web

Oque são Servers

(Voltar)

Os Servidores da web (web server) são responsáveis por hospedar o conteúdo do seu site, ou aplicação baseada na web para os usuários através da Internet. Podem ser imagens, textos, dados de aplicativos, vídeos, por exemplo.

Oque são as Tags

(Voltar)

As Tags HTML são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Por exemplo: a tag de parágrafo é escrita como < p > e < /p >". Já para negritar pode ser usada a tag < Strong> ou < b> no início e < /Strong> ou < /b> para finalizar. Para finalizar a utilização de uma tag pode ser usada uma barra (/), sendo que no caso das tags de autofechamento, não há necessidade da presença desse caractere.

Oque são listas Ordenadas e Não ordenadas?

Lista ordenada:

As listas ordenadas ou numeradas são usadas para indicar alguma sequência ou numeração

As tags usadas para criar uma lista não ordenada são "< ol >" e "< li >". A tag "< ol >" é usada para definir a lista e a tag < li > é usada para cada item da lista.

Não ordendada:

A tag < ul> é um elemento HTML que representa uma lista não ordenada. Este elemento tem como característica criar listas não ordenadas no HTML, ou seja, ao invés de apresentar uma lista com numeração a lista será apresentada com marcadores.
